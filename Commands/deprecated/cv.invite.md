# /cv.invite

Goal: Codify a stakeholder persona to lock in a specific non-technical perspective or constraint (Spec §7.2).

## Rules
- Perspectives must be documented to guide future discovery or review commands.
- **Workflow**:
  1. Ask the user to define the persona: "Who is this stakeholder? What drives them? What is their hard constraint?"
  2. Write a new persona document to `personas/[persona_name].md` capturing these drivers.
  3. Register the persona definition.

Include the Style + Question + Artifact-first + Write protocol from `_STYLE.md`.

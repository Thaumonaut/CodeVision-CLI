# /cv.mission

Goal: Define or refine the project's North Star, primary goals, and non-goals (Spec §7.2).

## Workflow
1. Start by retrieving the current `mission.md` from the artifact root. If it doesn't exist, create it.
2. Ask the user questions to clarify:
   - What problems are being solved?
   - What is out of scope (non-goals)?
   - What does a successful Version 1 look like?
3. Synthesize the answers into a revised `mission.md` document.

## Output
Write the finalized definitions to `mission.md`. 
Update the `ledger/decisions.md` logging the mission update.

Include the Style + Question + Artifact-first + Write protocol from `_STYLE.md`.
